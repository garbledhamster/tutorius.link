Keyboard Shortcuts Save 1 000+ clicks daily.

Launch & Switch
  • ⊞ → type “Edge” → Enter. Goal: open any app < 2 s.
  • ⊞ S → Universal Search.
  • Alt Tab → cycle windows.
  • Alt ` → cycle windows of current app.
  • ⊞ L → lock screen instantly.
Window & Desktop Control
  • ⊞ Z → Snap Layouts menu.
  • ⊞ ↕ (← →) → snap left/right.
  • ⊞ Ctrl D → new desktop.
  • ⊞ Ctrl ←/→ → switch desktops.
  • ⊞ Tab → Task View overview.
Pro Moves
  • ⊞ Shift S → Snip & Sketch overlay.
  • ⊞ V → Clipboard history.
  • ⊞ Alt R → quick screen-record (Xbox Bar).
  • ⊞ . → Emoji & Symbols.
  • Ctrl Shift Esc → Task Manager.

Touchpad Gestures Glide through Windows.

Tap

Two-finger tap → right-click menu.

Swipe

Three-finger swipe ↑ → Task View; ↓ → Desktop.

Edge Pull

Swipe from right edge → Notification Center.

Spread & Pinch

  • Pinch in → Taskbar Search.
  • Spread out → open Widgets.
  • Two-finger pinch / zoom → zoom in apps.

Advanced

  • Four-finger tap → open Action Center (custom).
  • Assign three-finger tap to Play/Pause in Settings.

Explorer Workflow Minutes now, hours saved later.

Quick Access Tidy-up

Explorer → Options → “Show frequently used folders” off.

Tags & Collections

Right-click file → Properties → Add tags → search by tag.

Power Rename

Select files → Shift F10 → PowerToys Rename for patterns.

Preview Pane

Toggle Alt P to peek PDFs, media without opening.

Tabs Mastery

Ctrl T new tab, Ctrl Tab cycle tabs, drag to new window.

OneDrive Moves Sync, protect, declutter.

Storage Sense + Files on Demand
  • Settings → System → Storage → enable Storage Sense.
  • OneDrive → Settings → “Files on Demand”.
Known Folder Move

OneDrive → Settings → Sync and Backup → Manage backup.

Shared Libraries

Teams Site or SharePoint Library → “Add shortcut to OneDrive”.

Personal Vault

Enable 2FA-protected vault for sensitive docs.

Password Monitor

Edge → Settings → Privacy → Password Monitor to leak-check creds.
